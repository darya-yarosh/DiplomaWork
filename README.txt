# Дипломный проект "Web-приложение по управлению студенческими проектами"
 
Открыть программный продукт в одной из IDE.

Проверить установленную версию NPM на своём ПК через терминал:

    npm -v

При отсутствии NPM версии установить его последнюю версию через терминал: 

    npm install npm@latest -g

Установка требуемых библиотек через терминал:

   npm install

Открыть в терминале папку проекта и запустить приложение в режиме разработчика:

    npm start
